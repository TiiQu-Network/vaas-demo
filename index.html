What this demo page provides
Three verification use cases with the same output format: verdict, verification density, contradiction score, units analyzed, and provenance links.

How demos differ from pilots
Demos use a small number of hardcoded Units for portability. Pilots retrieve Units programmatically, version them, and define deterministic rules up front for your statement set and your reporting scope.

What VaaS is for

VaaS evaluates the defensibility of individual sustainability statements before publication, using deterministic checks and provenance-backed evidence.

What VaaS is not

It is not a reporting workflow tool and it does not judge plausibility. It anchors statements to evidence and applies explicit deterministic rules.
